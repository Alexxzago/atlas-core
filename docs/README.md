# ATLAS

Atlas es un empleado comercial con IA para empresas.

## Objetivo

Permitir que una empresa conecte su WhatsApp y tenga un asistente inteligente capaz de:

- Responder consultas.
- Recordar conversaciones.
- Captar leads.
- Derivar a un humano.
- Hacer seguimiento automático.

Versión inicial enfocada en inmobiliarias.
